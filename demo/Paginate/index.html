Paginate

A library for simple infinite scroll and paginate in Firebase.

Basics

A Scroll or Paginate call wraps a Firebase reference and provides the same API. The reference is read-only, but works almost anywhere a Firebase reference would be appropriate for reading data, including libs like AngularFire. 
// create a Firebase reference
var fb = new Firebase('https://fbutil.firebaseio.com/paginate');

// create an infinite scroll reference
var scrollRef = new Firebase.util.Scroll(fb, 'number');

// append the next 25 items
scrollRef.scroll.next(25);

// listen normally
scrollRef.on('child_added', function(snap) {
  console.log('loaded record ' + snap.key());
});
